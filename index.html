CIS Room Documentation

The CIS room refers to MR.Eno's classroom itself.

This is the general documentation of the room.

Please update this documentation as the room changes and gets upgraded.

General Stuff

This Documentation will cover almost everything in the CIS room from a hardware and software perspective. There will also be a lot of specs lists here.

A specs list looks like this:

  • CPU (Central Processing Unit)
  • RAM (Random Access Memory)
  • GPU (Graphics Processing Unit)
  • Storage (Either Solid State or Hard Disk)
  • Motherboard (MoBo for short)
  • PSU (Power Supply Unit)

General Work Pcs

  • AMD Ryzen 5 1600x (AIO coolers in use are old and will need replacement)
  • 8GB or 16GB of DDR4 2133MHz single/dual channel RAM

    • Most of them are running single channel which needs to be upgraded.

  • Nvidia Low-profile GTX-1050 ti
  • 1TB 2.5" HHD (Needs to be replaced with SSD)
  • MSI B350 PC Mate
  • Thermaltake smart 80+ bronze 650w PSU

These are very mid tier budget systems, capable of light gaming and workloads. Use these primarily for testout/cisco work and feel free to make changes to them with permission from Mr. Eno.

If you want to do something or test something feel free to do so. This class has a lot of resources that are dedicated for your use. Most questions should be directed towards the L3's present in the class as they will generally be able to help with most issues/problems. Mr. Eno is a great resource if they can't help you. It is highly encouraged to do projects.
